This is the most basic form of Boosted: precompiled files for quick drop-in usage in nearly any web project. We provide compiled CSS and JS (boosted.*), as well as compiled and minified CSS and JS (boosted.min.*). source maps (boosted.*.map) are available for use with certain browsers' developer tools. Bundled JS files (boosted.bundle.js and minified boosted.bundle.min.js) include Popper, but not jQuery.
