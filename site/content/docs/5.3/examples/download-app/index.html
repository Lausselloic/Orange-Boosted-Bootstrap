Orange Design System Mobile Library Implementation

An App to experience the Orange Design System

The Orange Design System

The Design System from Orange provides the reference information for it's digital branding for both design and development resources. The goal is to ensure a consistent user experience across all Orange digital touchpoints thanks to an extensive library of design resources conforming to accessibility and eco-design engagement.

Orange Design System App

In order to share a common usage experience of this Design System, we created a mobile app for both iOS and Android OS. With this app:

  • A designer or digital service creator will be able to use these apps to experiment, in each OS, all the available assets and variants to adjust possible customization.
  • A developer will be able to view the corresponding code customization and have a ready built app code base that shows how to best use the assets, conforming to the expected accessibility standards, for faster and more qualitative integration.
  • Find all available elements in the code library
  • Experience the available interactions on each components and select the right customization
  • Save time and find the right code to use by selecting the right component variant and it's possible customization
  • All components have been implemented to follow the Orange Design System requirement in order to reinforce the brand consistancy
  • Available components have been tested with code example for their accessibility performance
  • The App is available on both iOS and android
  • The Orange Design System app and its SDK are available in an open-source

The code libraries references in the app can be found at:
