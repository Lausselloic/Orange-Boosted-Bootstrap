Boosted and Leaflet

Usage

Integrate Leaflet with the Boosted helpers and utilities.

Leaflet is not included in Boosted. Add it by including the CSS and JavaScript plugin manually, or using a CDN like so:


<link href="https://cdn.jsdelivr.net/npm/leaflet@1.7.1/dist/leaflet.css" integrity="sha256-BPfK9M5v34c2XP6p0cxVz1mUQLst0gTLk0mlc7kuodA=" crossorigin="anonymous" rel="stylesheet"/>
<script src="https://cdn.jsdelivr.net/npm/leaflet@1.7.1/dist/leaflet-src.min.js" integrity="sha384-Ozw1UodX5qtjpTA1yLoGz0jW69awt5rGfU87OIxh7l5BLcgqt9rAylprwTx+C2jF" crossorigin="anonymous" async></script>

Startups

{{< map.inline >}} {{ $length := len $.Site.Data.startups }} {{ $counter := 0 }} {{- range $.Site.Data.startups -}} {{- $counter = add $counter 1 -}} {{- $marker := . | jsonify -}} {{- $.Scratch.Add "markers" (slice $marker) -}} {{- end -}} {{ $markers := delimit ($.Scratch.Get "markers") "," }}
{{< /map.inline >}}

Accessibility

To ensure accessibility, the simplest way is to display an HTML list of markers alongside your map.

In last resort —and only if such a list should not be visually displayed— you may use .visually-hidden helper.

    {{< list.inline >}} {{- range $.Site.Data.startups }}
  • {{ .title | safeHTML }}
    {{ if .content }}

    {{ .content | safeHTML }}

    {{ end }}
    • {{- end -}} {{< /list.inline >}}
